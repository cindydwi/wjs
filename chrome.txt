Buka Chrome Remote Desktop
Klik Siapkan melalui SSH pada kolom sebelah kiri
Klik tombol Mulai
Login ke server melalui aplikasi SSH
Jalankan perintah ( sudo wget https://dl.google.com/linux/direct/chrome-remote-desktop_current_amd64.deb )
Selanjutnya install menggunakan perintah ( sudo apt-get install /home/usernamekalian/chrome-remote-desktop_current_amd64.deb )
Klik tombol beriktnya pada chrome remote desktop
Klik tombol izinkan
Copy Code Debian Linux dan pastekan ke server menggunakan ssh
Masukkan PIN
Tunggu hingga proses selesai
